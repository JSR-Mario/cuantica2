{\bfseries PageRank cuántico para el ciclo de 3 vértices. 
Utilizando la caminata cuántica continua del ejercicio anterior, defina el PageRank cuántico de cada vértice como el promedio temporal de la probabilidad: 
\begin{equation*}
QR_{i}=\lim_{T\rightarrow\infty}\frac{1}{T}\int_{0}^{T}P_{i}(t)dt
\end{equation*}}

\begin{enumerate}
    \item[a)] {\bfseries A partir de las expresiones obtenidas para $P_{i}(t)$ (ejercicio 2), calcule explícitamente $QR_{0}$, $QR_{1}$ y $QR_{2}$.}
    \item[b)] {\bfseries Verifique que $\sum_{i}QR_{i}=1$.}
    \item[c)] {\bfseries Interprete el resultado: ¿qué vértice resulta más importante según esta medida?}
\end{enumerate}
